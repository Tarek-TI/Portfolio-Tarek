Vue d'ensemble du projet

Ce projet a consisté en l'installation et la configuration de Microsoft SQL Server 2019 et Exchange Server 2019 sur un environnement virtualisé. J'ai également conçu et implémenté une base de données avec des tables, des vues et des droits d'accès spécifiques pour différents groupes d'utilisateurs.

Réalisations principales

Installation de SQL Server

  • • Installation d'une nouvelle instance de SQL Server 2019, nommée 'tp1'.
  • • Utilisation du mode d'authentification Windows seulement et ajout du compte Active Directory 'db_administrator'.

Requêtes et Bases de données

  • • Création d'une nouvelle base de données nommée 'maBase'.
  • • Création des tables DEPT et EMP avec des clés primaires et étrangères.
  • • Insertion des enregistrements de données pour chaque table.

Configuration des droits d'accès

  • • Configuration de l'instance pour se connecter par défaut à 'maBase'.
  • • Autorisation du groupe 'Managers' à modifier le contenu des tables, sans droit LDD.
  • • Autorisation du groupe 'Compta' à lire la vue 'VueEmp'.

Défis rencontrés et solutions

Gestion des privilèges

Définir des droits d'accès précis pour chaque groupe d'utilisateurs sur la base de données, en évitant de donner des privilèges excessifs.

Intégration Active Directory

S'assurer que l'instance SQL Server reconnaisse correctement les groupes et les utilisateurs de l'Active Directory pour l'authentification et les autorisations.
